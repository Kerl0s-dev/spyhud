# Spy Hud - Documentation

## Overview
**Spy Hud** is a lightweight and configurable HUD mod for Minecraft. It provides players with essential in-game information at a glance, along with a smooth and customizable zoom feature. The mod is designed to be modular, unobtrusive, and fully configurable via a simple GUI.

---

## Features

### Zoom
- **Custom Zoom Levels**: Choose from multiple zoom intensities (`x1.5`, `x2`, `x5`, etc.).  
- **Smooth Zoom**: Toggle smooth transition effects for a professional look.  
- **Hold to Zoom**: Optional feature to zoom only while holding the assigned key.  
- **Keybinds**: Activate Zoom, Zoom In / Zoom Out (adjust levels).

### HUD
- **Positioned HUD**: Displays essential info like player coordinates, current FPS, targeted block info.  
- **HUD Animations**: Multiple animations for components when appearing/disappearing: `Fade`, `Slide Left`, `Slide Right`, `Pop In`, `Pop Out`.  
- **Show / Hide HUD**: Easily toggle visibility with a keybind.  
- **Customizable Appearance**: Adjust colors, transparency, and animation style via the config menu.

---

## Configuration
The mod provides a clean GUI accessible in-game or via the mod menu.

**General Settings:**
- **Smooth Zoom**: Enable smooth transitions when zooming.  
- **Hold to Zoom**: Zoom only while holding the zoom key.  
- **Zoom Levels**: Select and customize the zoom intensity list.  
- **Display HUD**: Show or hide the HUD.  
- **HUD Animation Style**: Choose how HUD elements appear on screen.

---

## Installation
1. Install **Fabric Loader** compatible with your Minecraft version.  
2. Download the latest **Spy Hud .jar** file from Modrinth or GitHub releases.  
3. Place the `.jar` in your Minecraft `mods` folder.  
4. Launch Minecraft and configure your keybinds and settings in-game.
